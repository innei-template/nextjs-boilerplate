# NextJS boilerplate

This boilerplate port from [Shiro](https://github.com/Innei/Shiro)

The purpose is to quickly build a UI/UX prioritized frontend application.

## Tech Stack

- Jotai
- Framer Motion
- Radix Color
- Radix UI
- @next-ui/theme
- TailwindCSS
- TanStack Query
